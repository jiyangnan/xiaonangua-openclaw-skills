# 爆款文章框架 Skill

> 基于与 Claude Code 的讨论沉淀，适用场景：自媒体文章、公众号推文、博客长文

---

## 核心认知

**爆款不是"写得好"，而是"让人想读完 + 想转发"。**

这是两个不同的心理机制：
- 读完 → 情绪共鸣 + 认知价值 + 节奏驱动
- 转发 → 社交货币 + 身份认同 + 实用价值

---

## 爆款文章五要素

| 要素 | 作用 | 本质 |
|------|------|------|
| **共鸣** | 让人"对对对，就是这样" | 情绪共振 |
| **价值** | 读完后有实际收获 | 信息增量 |
| **节奏** | 像坐过山车，有起伏 | 情绪曲线 |
| **认知刷新** | 让人"原来如此" | 认知刷新 |
| **传播性** | 让人想转发 | 社交货币 |

## 经过验证的文章结构模型

### 模型 A：痛点 - 方案 - 验证（推荐 AI 写作使用）

```
[痛点场景化] → [为什么传统方法失效] → [新框架/方法] → [3个案例] → [立即可用的行动清单] → [升华/转发钩子]
```

### 模型 B：故事弧线

```
[冲突开场] → [困境升级] → [转折点] → [解决方案] → [新的平衡]
```
适合：个人成长、职业转型、创业故事

### 模型 C：金字塔结构

```
[结论先行] → [分论点1] → [分论点2] → [分论点3] → [行动建议]
```
适合：商业分析、方法论、教程

### 模型 D：SCQA（麦肯锡结构）

```
Situation（情景）→ Conflict（冲突）→ Question（问题）→ Answer（答案）
```

### 模型 E：好奇心循环（最适合自媒体）

```
[制造悬念] → [信息注入] → [再次制造悬念] → [闭合悬念] → [留新悬念]
```

---

## 混合模型（推荐用于情感/成长类主题）

结合 A + E：

```
[制造认知缺口] → [打破常识] → [提供新框架] → [案例验证] → [行动召唤] → [留白 + 转发钩子]
```

**关键：每个段落结尾都是下一个段落的诱饵**

---

## AI 辅助写作人味儿保持技巧

| AI 擅长 | 人来补充 |
|---------|----------|
| 框架搭建 | 个人故事/经历 |
| 信息整合 | 独特的情绪感受 |
| 语言优化 | 口语化表达 |
| 结构建议 | 停顿/留白 |

**具体技巧**：

1. **强制注入"不完美"** — 适度的重复、口头禅、感叹词
2. **用具体替代抽象** — "我那天在咖啡馆遇到..." 而不是 "很多人会发现..."
3. **保留认知过程** — "一开始我也以为...但后来发现..."
4. **声音指纹** — 每隔几段有一句"这就是我想说的..."

## 检查清单

### 开头（0-300字）

- [ ] 制造认知缺口/悬念
- [ ] 让读者觉得"这和我有关"
- [ ] 明确/暗示读完能获得什么

### 主体（300-1500字）

- [ ] 每 300 字有一个小高潮/转折
- [ ] 有具体案例/故事
- [ ] 有反常识观点
- [ ] 段落间有"钩子"连接

### 结尾（1500-2000+字）

- [ ] 有明确的行动建议
- [ ] 有转发钩子（身份认同/实用价值）
- [ ] 有留白，引发思考
- [ ] 有独特的"声音指纹"

### 整体

- [ ] 读起来像"一个人在说话"，不是"机器在输出"
- [ ] 有情绪起伏，不是平铺直叙
- [ ] 有具体细节，不是全是抽象概念

## 框架模板

```markdown
# [标题：制造悬念/好奇心]

## [开头：制造认知缺口]
- 场景化描述
- 痛点/问题引入
- 认知缺口制造

## [第一部分：打破常识]
- 反直觉观点
- 为什么会这样
- 传统方法的局限

## [第二部分：新框架/方法]
- 核心观点/方法论
- 为什么有效
- 底层逻辑

## [第三部分：案例验证]
- 3个具体案例
- 不同视角/人群
- 成功/失败对比

## [行动清单]
- 立即可用的 3-5 个行动点
- 最小可执行单元
- 效果预估

## [结尾：升华 + 转发钩子]
- 核心观点升华
- 身份认同触发
- 留白 + 互动召唤
```

---

## 使用方法

1. **选模型**：根据内容类型选择合适模型
2. **填框架**：按模板填充内容
3. **加钩子**：每个段落结尾加诱饵
4. **注入人味儿**：用具体故事/感受替换抽象表达
5. **检查清单**：逐项核对
