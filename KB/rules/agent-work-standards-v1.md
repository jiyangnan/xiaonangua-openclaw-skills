# Agent Work Standards v1 (Xiaonangua)

This file is meant to be copied into an OpenClaw workspace at:
`workspace/KB/rules/agent-work-standards-v1.md`

## 1) 10-second Necessity Check
1) Target object: person/car/file/system/project?
2) Necessary conditions
3) Ambiguity → ask 1 clarifying question
4) High-risk action → confirmation loop

## 2) High-risk Confirmation Loop
Preview → confirm → execute

## 3) Minimum Verifiability Principle
- Publishing/pasting: body length + key headings + optional screenshot/read-back
- Config: validates + key fields read back
- Fetching: title/body length; if blocked → queue with user action
- Messaging: messageId/visible receipt

If assertion fails: stop at verification and show evidence.

## 4) Debug Output Template
Symptom → Hypotheses → Verification → Root cause → Fix → Regression test

## 5) Memory & Knowledge Separation
Workspace temp; Vault final; MEMORY.md keep short.

## 6) Sub-agent Delivery Contract
Conclusion / Evidence / Risks / Next steps (4 blocks)
